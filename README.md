# 📘 Function Visualizer

## 🪪 프로젝트 개요

**프로젝트명:** Function Visualizer  
**주요 언어:** C++  
**그래픽 라이브러리:** SDL2  
**개발 목적:**  
C++와 SDL2를 이용하여 **수학 함수를 시각적으로 출력하는 프로그램**을 구현한다.  
직접 수학 연산 함수를 작성하고, 그래프를 실시간으로 그리며  
좌표 변환, 확대/축소, 구조적 아키텍처 설계를 학습한다.

---

## 🎯 주요 목표

1. **수학 함수를 시각화할 수 있는 프로그램 구현**
2. **Math 라이브러리를 직접 제작하여 기초 수학 계산 수행**
3. **C++ 객체지향 구조로 확장 가능한 그래픽 아키텍처 설계**
4. **좌표 변환, 축 그리기, 확대/축소 등의 시각적 인터페이스 제공**
